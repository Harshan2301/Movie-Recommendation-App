# CineScope Movie-Recommendation-App 

CineScope is a modern movie discovery web application that allows users to search movies, explore trending titles, view detailed information, and save favorite films.
The application uses the **TMDb API** to fetch real-time movie data.

---

## 🚀 Features

• 🔎 Search movies instantly
• 🎥 View movie posters, ratings, and details
• 📈 Browse trending and popular movies
• ⭐ Save movies to favorites collection
• 🎯 Personalized movie recommendations
• 📱 Fully responsive UI
• ⚡ Fast API-based movie data loading

---

## 🛠 Tech Stack

Frontend

## 🔑 Getting Started

### 1 Clone the repository

```
git clone https://github.com/yourusername/cinescope-movie-app.git
```

---

### 2 Open the project

Simply open the HTML file in your browser.

```
movie_recommendation_app.html
```

---

### 3 Get a TMDb API Key

Create an account on **TMDb** and generate an API key.

---

### 4 Enter API Key

When the app loads, paste your **TMDb API key** in the input box:

```
Paste TMDb API key here
```

Click **Save Key** to load movie data.

## 🎥 Application Features Explained

### Movie Search

Users can search for any movie by title.

### Trending Movies

Displays popular movies currently trending on TMDb.

### Movie Details

Shows:

• Poster
• Rating
• Release year
• Language
• Genre
• Overview

### Recommendations

Shows similar movies based on the selected film.

### Favorites

Users can bookmark movies and store them locally in the browser.
